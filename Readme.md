# Key, Button, & Axis Values for .binds Files

The following is a list of key values to be used in custom `.binds` files for
[Elite: Dangerous](https://www.elitedangerous.com/). These key values allow you to
edit your custom XML `.binds` file in a text editor, rather than having to boot the
game, sift through the options, make your binds, and apply.

Use this readme to find the Key Values, Joystick Buttons/Axes, and Xbox 360/Xbox 1
Button values you need to easily tweak your ship controls to suit you.

I couldn't find this list anywhere else online, so hopefully it's useful to you and
your wing, CMDR.
